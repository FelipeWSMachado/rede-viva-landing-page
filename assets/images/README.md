# Pasta de Imagens

Coloque aqui as imagens da landing page.

## Imagem Principal do Hero

**Arquivo:** `hero-background.jpg`

**Especificações:**
- Tamanho: 1920x1080px ou superior (4K recomendado)
- Formato: JPG ou WebP
- Peso: Máximo 1MB (otimizado)

**Como usar:**
1. Coloque a imagem aqui com o nome `hero-background.jpg`
2. Siga as instruções no arquivo `IMAGENS.md` na raiz do projeto

## Outras Imagens

Você pode adicionar outras imagens aqui conforme necessário:
- Logo: `logo.png` ou `logo.svg`
- Imagens de seções específicas
- Ícones (se necessário)

---

**Dica:** Use ferramentas como [TinyPNG](https://tinypng.com/) ou [Squoosh](https://squoosh.app/) para otimizar as imagens antes de usar.
